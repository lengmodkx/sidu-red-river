# 四渡赤水 · Four Crossings of the Red River

一部交互式历史可视化网站,把 1935 年 1 月 19 日至 3 月 22 日(共 73 天)中央红军四渡赤水的兵力调动、地形地势、行军路线、敌我态势,放到一张可拖动时间轴的地图上。

## 关键特性

### 1. 全程态势推演(Master View)
Hero 下方的主图,带可拖动日期滑块。可:
- **拖动时间轴**,红军位置在 5 个阶段关键帧之间线性插值(真移动,不是切帧)
- **累计叠加**已走过的行军路线,带方向箭头
- **累计标注**已发生的战斗节点
- **自动播放** 0.5× / 1× / 2× 倍速,跑完整场战役
- 实时显示当前日期 + 所属阶段

### 2. 阶段详图(每阶段)
5 个阶段(战前 + 一/二/三/四渡)各有:
- 部队框(18px 高,仅番号)
- 锚点小三角(指示部队钉在哪座城)
- 自动错位排布(同地点的多个番号向上堆叠,不再叠成一片)
- 红色行军路线 `pathLength` 动画绘制
- 国军反应式路线虚线淡入
- 部队逐个 spring 进场

### 3. 历史细节
每个阶段附带:
- 战略背景叙事
- 战略点评(毛泽东的腾挪之道)
- 关键事实卡片(时间、渡口、关键战役、战果)

## 技术栈

- **Vite 8** + **React 19** + **TypeScript**
- **framer-motion 12**(路径绘制、布局动画、入场 stagger)
- SVG 矢量地图(山脉、河流、省界、渡口)

## 启动

```bash
npm install
npm run dev
```

打开 http://127.0.0.1:5173/ 。

## 构建

```bash
npm run build
```

## 数据来源

- 部队番号、指挥员、兵力:历史教材通行说法
- 地理位置:经纬度映射到 1000×700 SVG 视口(中国黔北、川南、滇东北)
- 路线 / 战斗节点:出自《中国工农红军长征史》等公开资料

## 目录结构

```
src/
  data/
    places.ts    # 50+ 真实地名(遵义、桐梓、土城、扎西、皎平渡...)
    forces.ts    # 5 红军番号 + 13 国军番号(番号、指挥、兵力)
    phases.ts    # 5 阶段数据 + 时间插值工具
  components/
    Map.tsx           # 阶段详图
    MasterMap.tsx     # 主图(可拖动日期)
    MasterView.tsx    # 主图容器 + 时间轴控件
    Section.tsx       # 阶段页
    Timeline.tsx      # 顶部阶段导航
    Hero.tsx          # 首屏
    Recap.tsx         # 复盘
```
